# 数字匹配游戏 (Memory Game)
一款考验记忆力与观察力的数字匹配小游戏，通过翻牌找到相同数字对来完成挑战。
## 游戏介绍
数字匹配游戏是一款经典的记忆类游戏，玩家需要翻开卡片寻找相同数字的配对。游戏包含多种难度级别，记录玩家的最佳成绩，并支持游戏过程回放功能。
## 功能特点
1. 多种难度级别：22、33、44、46 四种不同规模的游戏板
2. 计时功能：记录游戏用时
3. 步数统计：记录玩家完成游戏的总步数
4. 成绩记录：保存最佳游戏记录（最少时间和最少步数）
5. 步骤记录：保存每局游戏的详细步骤，支持回放
6. 音效反馈：点击、匹配和胜利时的音效提示
7. 简洁美观的图形界面
## 游戏玩法
选择难度级别开始游戏
点击卡片翻开数字
寻找相同数字的卡片进行配对
所有卡片配对成功即完成游戏
系统会记录你的用时和步数，挑战最佳记录
## 项目结构
MemoryGame/
- ├── Function.py        # 核心功能模块（数组生成、计时等）
- ├── BestRecord.py      # 最佳记录管理模块
- ├── RecordSteps.py     # 游戏步骤记录模块
- ├── TwoPlayerMode.py   # 双人模式相关功能
- ├── UI.py              # 游戏界面实现
- ├── best_records.json  # 最佳记录数据文件
- ├── step_records.json  # 游戏步骤记录数据文件
- ├── mus/               # 音效文件目录
- │   ├── click.mp3      # 点击音效
- │   ├── match.mp3      # 匹配成功音效
- │   └── victory.mp3    # 胜利音效
- └── img/               # 图片资源目录
-    └── number.png     # 游戏图标
## 核心模块说明
1. Function.py
生成随机数字数组（游戏卡片数据）
步数计数
时间计算与格式化
2. BestRecord.py
初始化最佳记录文件
读取和更新最佳记录
按难度级别管理记录
3. RecordSteps.py
记录每局游戏的详细步骤
保存和读取步骤记录
支持按难度查询最新记录
4. UI.py
基于 PyQt5 的图形界面实现
游戏卡片交互逻辑
音效播放
游戏状态显示（时间、步数）

## 记录说明
最佳记录保存在best_records.json文件中
每局游戏的详细步骤保存在step_records.json文件中
记录包含时间、步数、难度和每一步的操作信息
